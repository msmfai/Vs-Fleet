# Support

Fleet is pre-release alpha software.

## Alpha support boundary

- Support is best-effort.
- Breaking changes are expected.
- There are no production support guarantees, response SLAs, paid support
  terms, or stable release lines yet.
- Source builds and local macOS dogfooding are the intended alpha path.
- Public binary distribution, remote/container deployment, auto-update, and
  packaged extension publishing are not supported alpha commitments yet.

## Where to ask

Use GitHub issues for reproducible local-alpha bugs or release-readiness
feedback. Do not post credentials, private logs, unredacted screenshots, local
URLs, or workspace paths.

Security vulnerabilities should follow `SECURITY.md`, not public issues.
