# Name Collision Review

Status: no trademark clearance claim.

Known collision: JetBrains has used `Fleet` for a developer IDE/product.
Source checked: <https://www.jetbrains.com/fleet/>

Current release posture: `Fleet` is a provisional source-alpha working name.
The repository makes no trademark claim to the name. Stable package or binary publication under Fleet namespaces is deferred until the owner either clears the name or renames it.

Owner decision source: `docs/release/OWNER_DECISION_RECORD.md`, section
`Public Name Collision And Trademark Posture`.
