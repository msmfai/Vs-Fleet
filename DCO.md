# Developer Certificate of Origin

Fleet requires Developer Certificate of Origin (DCO) sign-off on outside code
contributions.

Add this line to every commit message:

```text
Signed-off-by: Your Name <your.email@example.com>
```

The easiest way to add it is:

```sh
git commit -s
```

By adding a `Signed-off-by` line, you certify that you wrote the contribution
or otherwise have the right to submit it, that you can license it under the
project license, and that the sign-off may be kept in the public project
history.

For AI-assisted work, you also remain responsible for the submitted code and
must not include private prompts, private model transcripts, private logs,
workspace paths, generated build outputs, raw logs, or machine-specific paths.
