# Research

Source-controlled research dossiers for Fleet the designs.

These are not scratch notes. Each dossier should capture:

- upstream facts with links to primary sources where possible;
- what those facts imply for Fleet's current architecture;
- hypotheses that still need VM, container, or GUI verification;
- concrete follow-up experiments.
